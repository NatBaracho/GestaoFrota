# 🚗 Gestão de Frota

Aplicação Windows Forms em C# para gerenciamento de veículos de uma frota.  
Permite cadastrar, listar e remover veículos, diferenciando entre **Carros** e **Caminhões**, com cálculo automático do custo de aluguel.

---

## 📋 Funcionalidades

- Cadastro de veículos com informações:
  - Placa
  - Modelo
  - Ano
  - Valor da diária
  - Específico (Qtd. de portas para carros ou capacidade de carga para caminhões)
- Listagem dos veículos cadastrados em uma tabela interativa
- Remoção de veículos selecionados
- Cálculo automático do custo de aluguel para um período definido

---

## 🛠️ Tecnologias utilizadas

- **C#** (.NET Framework / Windows Forms)
- **BindingSource** para vinculação de dados
- **Repository Pattern** para manipulação dos veículos
- **DataGridView** para exibição dos dados
